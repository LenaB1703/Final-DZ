Git

  • git init - создание нового репозитория Git в пустой директории или превращение существующий в новый репозиторий.
  • git clone - клонирование репозитория из удаленного хранилища на локальную машину.
  • git push - выгрузка сделанных изменений в удаленный репозиторий.
  • git pull - получение изменений в удаленный репозиторий.
  • git status - просмотр поиска измененных файлов
    • git branch - создание ветки git checkount - переход на ветку git commit - сохранение изменений в Git git log - просмотр истории коммитов в репозитории
